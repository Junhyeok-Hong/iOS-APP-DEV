iOS application designed to streamline habbit formation.

The user is able to create a new habbit with options:
- Start Date
- End Date
- Repeating days

Then for each date, the user will be prompted with their list of habbits.
When the user completes a habbit, they are able to mark it "complete".

When the user swipes a selected habbit to the left, they are prompted with the following options:
- Delete the habbit
- View habbit statistics

The habbit statiscs page displays:
- Total success rate
- Current month's success rate
- Longest streak
- Remaining days
